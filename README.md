✨ Features

⬇️ 100+ Platforms — YouTube, Instagram, TikTok, Twitter/X, Vimeo, SoundCloud and more

🎬 Quality Selection — 4K to 240p, MP4, WebM, MP3, OGG, WAV

🎵 Audio Mode — Download audio as MP3 from any video

🏅 29 Achievements — Steam-style achievement system, XP and level

🎨 7 Themes — Dark, Neon, Crimson, Gold, Ice, Purple, Light

🌍 8 Languages — Turkish, English, German, Spanish, Japanese, Korean, French, Chinese

👤 Account System — Profile, statistics, login log

📚 Library & History — Downloads are automatically saved

⚡ Speed Indicator — Instant download speed and progress

🔊 Sound Effects — Achievements and downloads Sounds

🔑 Developer Mode — Exclusive developer access

📱 Mobile Compatible — Works seamlessly on every device

🚀 Usage
Open JUG
Paste the video URL
Select quality and format
Click the download button ✅
